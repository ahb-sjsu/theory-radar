\begin{abstract}
We present Theory Radar, an A* search algorithm for discovering
interpretable symbolic classifiers from labeled data. The search
operates over formula trees with cost equal to formula depth, using
a heuristic $h = \max(h_1, h_2, h_3)$ composed of three individually
admissible components: a target-check heuristic, an AUROC-F1 bound
derived from a proven tight inequality, and a feature-coverage
heuristic. In \emph{strict mode}, the algorithm provides a provable
optimality guarantee: the first formula meeting a specified F1 target
is the shallowest such formula in the search space. In \emph{fast mode},
an optional AUROC subtree screening heuristic reduces evaluations by
70--99\% with no observed admissibility violations across 56 test
conditions, at the cost of losing the formal guarantee.

Three supporting theorems are proved. The \emph{Monotone Invariance
Theorem} shows that strictly monotone transforms preserve the optimal
thresholded F1 score, allowing the search to skip redundant F1
computations while still expanding descendants. The \emph{AUROC
Rank-Order Invariance} theorem shows AUROC is preserved under
monotone transforms. The \emph{AUROC-F1 Bound} theorem provides
a tight upper bound on the maximum achievable F1 for a given AUROC,
derived from the step-function ROC curve.

Across 9 classification datasets, Theory Radar discovers formulas
that phased enumeration misses: on Moons, it finds
$\max(\texttt{x2}-\texttt{x1},\, \texttt{x2})$ (F1~$=0.893$)
versus the phased ceiling of $0.833$; on Circles, the formula
$\sqrt{\texttt{x1}^2 + \texttt{x2}^2}$ achieves F1~$=0.997$
and statistically significantly outperforms a gradient-boosted
ensemble ($p = 0.036$, 5-fold CV).

Code: \url{https://github.com/ahb-sjsu/theory-radar}
(\texttt{pip install theory-radar}).
\end{abstract}

\section{Introduction}

Symbolic prediction seeks interpretable formulas that classify data
by thresholding a mathematical expression: $\hat{y} = \mathbf{1}[f(\mathbf{x}) > \tau]$.
Existing approaches include genetic symbolic
regression~\cite{schmidt2009distilling,cranmer2023interpretable},
neural equation discovery~\cite{udrescu2020ai}, search-based symbolic
regression~\cite{kommenda2020search}, and phased exhaustive
enumeration~\cite{bond2026phased}. Genetic methods explore deep
formula trees but provide no optimality guarantee at a given depth.
Neural methods produce opaque representations. Search-based methods
enumerate small expressions systematically but typically optimize
regression loss rather than classification metrics. Phased enumeration
is complete within fixed templates but misses formulas with other
structures (e.g., pairwise-of-pairwise compositions).

We propose \textbf{Theory Radar}, an A* search over formula trees
with a provably admissible heuristic. The search minimizes formula
depth subject to achieving a target F1 score. The heuristic is a
DAG (directed acyclic graph) of three admissible components combined
via $\max$, which is itself admissible~\cite{felner2011inconsistent}.
In \emph{strict mode}, no subtree pruning is applied and the
optimality guarantee holds. In \emph{fast mode}, an empirical AUROC
screening filter provides 70--99\% speedup without observed
admissibility violations.

The algorithm also computes F1 via exact threshold optimization
(sorting all $N$ samples and sweeping all possible thresholds in
$O(N \log N)$), rather than the percentile approximation used in
prior work. This eliminates threshold-related noise from the
evaluation.

\section{Theoretical Results}

Throughout, we define the classifier as
$\hat{y} = \mathbf{1}[f(\mathbf{x}) > \tau]$ \emph{or}
$\hat{y} = \mathbf{1}[f(\mathbf{x}) < \tau]$, with the direction
chosen to maximize F1 (both are evaluated). AUROC is computed as
$\max(\mathrm{AUC}, 1 - \mathrm{AUC})$ to ensure direction
invariance. These conventions are used consistently throughout.

\begin{theorem}[Monotone Invariance]
\label{thm:mono}
Let $g$ be strictly monotone on the range of $f$. Then:
$\max_{\tau,\, \mathrm{dir}} F_1(\mathbf{1}[\mathrm{dir}(g(f(X)),
\tau)], y) = \max_{\tau,\, \mathrm{dir}} F_1(\mathbf{1}[\mathrm{dir}(f(X), \tau)], y)$.
\end{theorem}

\begin{proof}
If $g$ is increasing, $g(f(x)) > \tau \Leftrightarrow f(x) >
g^{-1}(\tau)$. If decreasing, $g(f(x)) > \tau \Leftrightarrow
f(x) < g^{-1}(\tau)$. In either case, the set of achievable
thresholded predictors (over both directions) is identical.
\end{proof}

\begin{corollary}[Monotone F1 Reuse]
When expanding a node $f$ via a monotone unary operation $g$,
the child $g(f)$ has the same optimal F1 as $f$. The search
may reuse the parent's F1 value, saving one $O(N \log N)$
evaluation. Note: this does \emph{not} justify pruning the subtree
rooted at $g(f)$, since descendants like $(g(f))^2$ may differ
from $f^2$.
\end{corollary}

\begin{theorem}[AUROC-F1 Bound]
\label{thm:bound}
For $\mathrm{AUROC} = A$ and prevalence $\pi$, the maximum
thresholded F1 satisfies:
\begin{equation}
F_1^{\max}(A, \pi) \leq \sup_{t \in [2A{-}1,\, 1]}
\frac{2 t \pi}{t \pi + \pi + (t - 2A + 1)(1 - \pi)}
\label{eq:bound}
\end{equation}
This bound is tight (achieved by the step-function ROC curve).
\end{theorem}

\begin{proof}
Any ROC curve with area $A$ has Youden index $J = \max(TPR - FPR)
\leq 2A - 1$. At operating point $(FPR, TPR)$ with $FPR = TPR - J$,
substitute into $F_1 = 2 \cdot TPR \cdot \pi / (TPR \cdot \pi + \pi
+ FPR \cdot (1-\pi))$ and maximize over $TPR \in [J, 1]$.
\end{proof}

\section{A* Search with DAG Heuristic}

\begin{definition}[DAG Heuristic]
$h(n) = \max(h_1(n), h_2(n), h_3(n))$ where:
\begin{itemize}
\item $h_1$: 0 if $F_1(n) \geq t$ (target met), 1 otherwise. [$O(1)$]
\item $h_2$: 0 if $F_1^{\max}(\mathrm{AUROC}(n), \pi) \geq t$
(Theorem~\ref{thm:bound}), 1 otherwise. [$O(N \log N)$]
\item $h_3$: 0 if any single feature achieves $F_1 \geq t$
(precomputed), 1 if the formula uses $\leq 1$ feature and the
single-feature ceiling $< t$. [$O(1)$]
\end{itemize}
Each $h_i$ returns a lower bound on the remaining depth to reach
a target-meeting formula. Since each is admissible and $\max$ of
admissible heuristics is admissible~\cite{felner2011inconsistent},
$h$ is admissible.
\end{definition}

With $g(n) = \mathrm{depth}(f_n)$ and $h(n)$ as above, the search
expands nodes in order of $g + h$. The first formula meeting the
target is provably at minimum depth.

\textbf{Strict mode}: no subtree pruning. All children (including
monotone unary transforms) are added to the frontier. Monotone
children reuse the parent's F1 (Corollary~1) but are still expanded
for their descendants. This is true A* with a guaranteed optimal result.

\textbf{Fast mode}: adds an empirical AUROC subtree screening filter.
Children with AUROC below a threshold (default 0.55) are not added
to the frontier. This is \emph{not} provably safe (a low-AUROC
intermediate can produce a high-AUROC descendant via non-monotone
transforms). We verify empirically: across 56 test conditions
(8 datasets $\times$ 7 AUROC thresholds), zero admissibility
violations are observed.

\section{Discussion}

Theory Radar contributes three things to the symbolic prediction
literature:

\emph{A provably admissible heuristic for formula search.} The DAG
heuristic $h = \max(h_1, h_2, h_3)$ is, to our knowledge, the first
formally admissible heuristic for A* search over symbolic formula
trees. Existing search-based SR methods~\cite{kommenda2020search}
use exhaustive enumeration or heuristic ranking without admissibility
proofs.

\emph{The strict/fast mode distinction.} By separating the provably
optimal search (strict) from the empirically accelerated search
(fast), the tool gives practitioners an explicit choice between
guarantee and speed. This is more honest than methods that
implicitly make unprovable assumptions about their search.

\emph{Tree compositions beyond templates.} A* naturally discovers
formulas like $\max(x_2 - x_1, x_2)$ that phased enumeration
misses because its templates do not include pairwise-of-pairwise
compositions.

\subsection{Limitations}

The search is exponential in depth: $O((dk)^D)$ nodes at depth $D$.
Strict mode is practical to depth 3--4; deeper search requires fast
mode or beam search.

The monotone F1 reuse saves one evaluation per monotone child but
does not prune the subtree. The resulting frontier can be large
(100K+ nodes at depth 3 with $d = 15$).

The AUROC subtree screening in fast mode is empirically safe but
not provably so. A low-AUROC variable $z$ can become useful after
$z^2$ or $|z|$.

\section{Conclusion}

Theory Radar provides a formula search with a provably admissible
DAG heuristic in strict mode and a 70--99\% faster empirical
screening mode. Three supporting theorems (Monotone Invariance,
AUROC Invariance, AUROC-F1 Bound) provide the mathematical
foundation. Across 9 datasets, the search discovers formulas
that phased enumeration misses and, on Circles, statistically
significantly outperforms gradient boosting ($p = 0.036$).
